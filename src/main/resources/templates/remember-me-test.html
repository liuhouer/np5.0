技术实现：

                    
    
                        
  • 基于 Spring Security RememberMe 机制
    • 
                        
  • 令牌存储在 Redis 中，支持集群部署
    • 
                        
  • 采用 Series-Token 模式，安全性高
    • 
                        
  • 令牌有效期 30 天，自动过期
    • 
                        
  • 每次自动登录会刷新 Token，防止令牌被盗用
    • 
                        
  • 退出登录时自动清除 Redis 中的令牌
